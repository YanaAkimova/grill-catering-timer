# 🔥 Grill Catering Timer — Облачный таймер для кейтеринга на гриле

Интерактивный таймер для синхронизации меню, расписания подачи блюд и рецептов на мероприятиях с грилем. Разработан для команды (шеф-повара, бankетного менеджера и организатора).

## 📋 Вкладки приложения

### 1. **Menu** (Меню)
Полный список всех блюд на мероприятии с параметрами готовки.

**Текущее меню:**
- Закуски (овощи, хлеб, масло)
- Салат с вырезкой
- Ribeye 300g
- Груша на гриле

### 2. **Timeline** (Расписание подачи)
Визуальное отображение времени готовки каждого блюда. Показывает, когда начать готовить и когда подавать.

### 3. **Team** (Команда)
Список участников и их роли:
- **Илья** — Шеф-повар (готовит на гриле)
- **Анна** — Бankетный менеджер (координирует подачу)
- **Роман** — Организатор (синхронизирует таймер)

### 4. **Recipes** (Рецепты)
Подробные инструкции по готовке каждого блюда с параметрами гриля.

## ⚙️ Параметры готовки

| Блюдо | Температура | Время | Сложность |
|-------|-------------|-------|-----------|
| Закуски | 80°C | 2-3 мин | Низкая |
| Салат с вырезкой | 85°C | 8-10 мин | Средняя |
| Ribeye 300g | 90°C | 10-12 мин | Высокая |
| Груша на гриле | 75°C | 4-5 мин | Низкая |

## 🚀 Быстрый старт

### Установка
```bash
npm install
```

### Развитие (Development)
```bash
npm run dev
```
Приложение откроется на `http://localhost:5173`

### Production Build
```bash
npm run build
```
