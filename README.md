# 🔥 Grill Catering Timer — Облачный таймер для кейтеринга на гриле

Интерактивный таймер для синхронизации меню, расписания подачи блюд и рецептов на мероприятиях с грилем. Разработан для команды (шеф-повара, банкетного менеджера и организатора).

## 📋 Описание

**Событие:** Кейтеринг на гриле «Академия гриля» — 26 апреля 2026, 13:00–18:00  
**Гости:** 8 человек  
**Команда:**
- 👨‍🍳 **Илья** — шеф-повар (готовит на гриле)
- 👩‍💼 **Анна** — банкетный менеджер (сервировка, напитки)
- 👨‍💻 **Роман** — организатор (координация, закупки)

Таймер показывает **одинаковое время** на всех устройствах и указывает:
- Активное блюдо (пульсирующая карточка)
- Когда начать готовить
- Когда подать гостям
- Рецепты с температурами

## ⚙️ Синхронизация времени

Все три устройства (Илья, Анна, Роман) показывают **одинаковое время**, основанное на серверном времени. Таймер синхронизируется автоматически.

**Перед событием (25 апреля):**
1. Все трое открывают таймер на своих устройствах
2. Проверяют, что время совпадает
3. Илья открывает вкладку **"Рецепты"** для справки
4. Анна следит за **"Таймлайном"** для сервировки
